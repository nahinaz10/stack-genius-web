# 🧮 Stack Genius — Web Based Stack Calculator

A fully interactive web-based calculator built with **HTML, CSS, C Program and JavaScript** that visually demonstrates how a **Stack data structure** works in real time.

Built as a Data Structures project to show the real-world application of stacks through an interactive UI.

---

## 🌐 Live Demo

> Open `stackgenius.html` in any browser — no installation needed!

---

## ✨ Features

- 🖩 **Calculator Module** — Full working calculator with button UI
- 🔄 **Conversion Module** — Converts Infix → Postfix with step-by-step trace
- 📊 **Stack Visualizer** — Watch the stack grow and shrink in real time
- ✅ **Input Validator** — Validates expressions before processing
- 📱 **Responsive Design** — Works on mobile and desktop
- 🎨 **Dark Theme UI** — Clean modern interface

---

## 🛠️ How It Works

```
Infix Expression  →  [Conversion Module]  →  Postfix Expression
Postfix Expression  →  [Evaluation Module]  →  Result
```

**Example:**
```
Input  : 3+5*2
Postfix: 352*+
Output : 13
```

## 🧠 Concepts Used

| Concept | Usage |
|---|---|
| Stack (Array-based) | Core data structure |
| Infix to Postfix conversion | Shunting-yard algorithm |
| Postfix Evaluation | Stack-based evaluation |
| Operator Precedence | `*`, `/` before `+`, `-` |
| DOM Manipulation | Real-time UI updates |
| Event Handling | Button clicks and keyboard input |
