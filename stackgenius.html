ABOUT THIS PROJECT
Project Overview

What is this?

This is a Data Structures project demonstrating the Stack data structure in C. A stack is used to convert infix expressions to postfix and then evaluate them — the core of any mathematical expression parser.

Algorithm — Infix to Postfix

  • Read the infix expression left to right
  • If operand → directly add to postfix output
  • If '(' → push to stack
  • If ')' → pop and output until '(' is found
  • If operator → pop higher/equal precedence operators first, then push
  • At end → pop all remaining from stack to output

Precedence Rules

+ - → precedence 1 (low)
* / → precedence 2 (high)
( ) → grouping (not an operator)

Modules in the C Program

  • Stack Module — push, pop, peek, isEmpty
  • Operator Module — isOperator, precedence
  • Conversion Module — infixToPostfix
  • Evaluation Module — evaluatePostfix
  • Main Module — input/output
